# Data Collection

[Podman Desktop](https://podman-desktop.io/) uses telemetry to collect anonymous usage data in order to identify issues and improve our user experience. 
You can read more in [podman-desktop/podman-desktop#USAGE_DATA.md](https://github.com/podman-desktop/podman-desktop/blob/main/USAGE_DATA.md).

The Podman Desktop Hummingbird extension extends the telemetry by providing special events related to the extension usage. 

## What's included in the telemetry data

- User pulling a Hummingbird image from the extension catalog.
- User opening a Hummingbird repository in the browser.
